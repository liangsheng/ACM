# 题目描述

输入整数 $N$，求出斐波那契数列中的第 $N$ 项是多少。

斐波那契数列的第 $1$ 项是 $1$，第 $2$ 项是 $1$，从第 $3$ 项开始的每一项都等于前两项之和。

前 7 个斐波那契数是 $[1, 1, 2, 3, 5, 8, 13]$

# 输入格式

共 1 行，包含一个整数 $N$

# 输出格式

输出 1 行，包含第 $N$ 个斐波那契数

```input1
7
```

```output1
13
```

```input2
50
```

```output2
12586269025
```

```input3
100
```

```output3
354224848179261915075
```

# 提示

# 数据规模与限制
* $3 \le N \le 1000$
