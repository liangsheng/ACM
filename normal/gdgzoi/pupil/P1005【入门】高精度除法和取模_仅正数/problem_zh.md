# 题目描述

给定两个非负整数（不含前导 $0$） $A，B$，请你计算 $A / B$ 的商和余数。

# 输入格式

共两行，第一行包含整数 $A$，第二行包含整数 $B$。

# 输出格式

共两行，第一行输出所求的商，第二行输出所求余数。

```input1
7
2
```

```output1
3
1
```

# 提示

# 数据规模与限制
* $1 \le A的长度 \le 100$,  
* $1 \le B \le 1000$,  
* $B$ 一定不为 $0$
