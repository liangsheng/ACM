# 题目描述

给定一个长度为 $n$ 的数列，求数值严格单调递增的子序列的长度最长是多少。


# 输入格式

第一行包含整数 $n$。

第二行包含 $n$ 个整数 $a_i$，表示完整序列。


# 输出格式

输出一个整数，表示最大长度。

```input1
7
3 1 2 1 8 5 6
```

```output1
4
```

# 提示
**【样例 #1 解释】**
* $[1, 2, 5, 6]$ 是长度为 $4$ 的上升子序列
* 本题的 $n$ 不超过 $1000$，所以 $O(n^2)$ 的算法是可以通过的
* $dp[i]$ 表示以 $a[i]$ 结尾的最长上升子序列的长度
